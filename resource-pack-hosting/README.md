# ITPlugin Resource Pack

This directory hosts the Minecraft resource pack for ITPlugin.

## Files

- `ITPlugin-ResourcePack.zip` - The actual resource pack ZIP file (you need to add this)
- `index.html` - Landing page with download link and SHA1 hash

## SHA1 Hash

The SHA1 hash of the resource pack is: `9972e33cfa916c26dcf4ba4b8deb0ab3dbd3f1e7`

## Adding the Resource Pack

Place your actual `ITPlugin-ResourcePack.zip` file in this directory before deploying.
